The Challenge

Vehicles Are Now Software‑Defined

Modern vehicles carry dozens of ECUs, each running multiple applications at different software versions. Rolling out an update is far more than sending a file over the air — it requires absolute control.

The Ecosystem

The 4‑Step Flow

VISTA doesn't work alone. It is part of a tightly integrated ecosystem that takes a business rollout decision all the way down to execution inside the vehicle.

1 Cosmos RM

Rollout Manager

Business Decision Layer
  • Defines rollout: What, Who, When
  • Selects software version
  • Targets vehicle group
  • Sets go-live schedule
2 VISTA

Orchestration Layer

Intelligent Control Plane
  • Converts intent → ECU logic
  • Tracks VIN & ECU state
  • Generates DRM
  • Manages consent
  • Emits status back to RM
3 V2C Hub

Vehicle ↔ Cloud

Secure Communication Bridge
  • Secure message exchange
  • Routes to correct ECU
  • Delivers ProvisioningPush
  • Returns DiscrepancyCheck
  • Reports Consent status
4 AOTA

Vehicle Execution

On-Vehicle Runtime
  • Receives deployment descriptor
  • Downloads software
  • Applies update
  • Reports final status upstream
